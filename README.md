# Online Voting System (DSA Project)

## Description
This project implements an **Online Voting System** using core **Data Structures and Algorithms**.  
It allows users to vote for candidates, prevents duplicate voting, counts votes efficiently, and displays election results in real time.

---

## Features

- Add candidates
- Cast votes
- Prevent duplicate voting
- Count votes efficiently
- Display winner and leaderboard

---

## Data Structures Used

- Hash Map (Dictionary) for vote counting
- Set for preventing duplicate votes
- Heap (Priority Queue) for ranking candidates

---

## Core Operations

- Add candidate
- Vote for candidate
- Get vote count
- Determine winner
- Display leaderboard
